# دليل التثبيت والمعايير

دليل سريع لإعداد بيئة التطوير والتوثيق.

## تثبيت التوثيق
```bash
pip install mkdocs mkdocs-material
```

## تشغيل موقع التوثيق محلياً
```bash
mkdocs serve
```

## معايير الكود (Coding Standards)
*   **اللغة:** العربية الإلزامية للتعليقات، السجلات، ورسائل الخطأ.
*   **التزامن:** استخدام `select_for_update` لجميع عمليات الكتابة في المخزون.
*   **المالية:** الالتزام الصارم بـ `COSTING_STRICT_MODE`.
