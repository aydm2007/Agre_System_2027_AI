# منطق الخدمات ودورة التكلفة

يتم تجميع التكاليف وتحويلها إلى أصول أو مصاريف بدقة متناهية.

## دورة حياة التكلفة
1.  **التجميع:** رصد المواد، الآلات، والعمالة في `Activity`.
2.  **الاشتقاق:** حساب القيم المالية عبر `calculate_activity_cost`.
3.  **التثبيت:** إنشاء `ActivityCostSnapshot` لتجميد الحالة المالية.

## التتبع الفوري (Traceability)
كل حركة مخزنية يقابلها قيد مالي فوري، مما يوفر رؤية أوضح للتدفقات النقدية والاصول البيولوجية.
