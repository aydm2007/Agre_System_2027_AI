# معايير التطوير (Protocols)

هذه المعايير غير قابلة للتفاوض لضمان علامة **100/100** في التدقيق الجنائي.

## البروتوكول الذهبي (Golden Rules)
1.  **نظافة قاعدة البيانات:** لا يسمح بالجداول الميتة (Zombie Tables).
2.  **النزاهة المالية:** الثبات المطلق للسجلات المالية.
3.  **التعريب:** النظام عربي بالكامل (داخلياً وخارجياً).

!!! info "مبدأ العيون الأربع"
    لا يمكن لمدخل بيانات أن يعتمد سجله الخاص؛ الموافقة تتطلب دائماً إشرافاً مستقلاً.
