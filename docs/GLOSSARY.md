# قاموس المصطلحات — AgriAsset (YECO Edition)
# Glossary — AgriAsset (YECO Edition)

> [!NOTE]
> هذا القاموس هو المرجع الرسمي لمصطلحات النظام. أي مصطلح غير مُعرّف هنا يجب إضافته قبل استخدامه في الوثائق.

---

## المصطلحات المالية

| المصطلح | الترجمة | التعريف |
|---------|---------|---------|
| **Surra (الصُرّة)** | وحدة أجر يومية | وحدة قياس العمالة في المزارع اليمنية — تمثل حصة عامل من يوم عمل كامل. ربع اليوم = `0.25` صرة. الأجر = `عدد الصُرر × معدل اليوم`. لا تُستخدم الساعات في الحساب المالي. |
| **WIP (1400)** | العمل تحت التنفيذ | حساب يجمّع تكاليف المحصول (مواد + عمالة + آلات) حتى الحصاد أو الإقفال الموسمي. |
| **COGS (6000)** | تكلفة البضاعة المُباعة | يُحمّل عند البيع أو الإقفال الموسمي من رصيد WIP. |
| **Fund Accounting** | المحاسبة القائمة على الصناديق | إيرادات المزرعة عهدة (Custodial) — ليست ملكاً للمزرعة. تُرحّل للقطاع. المصروفات من تمويلات معتمدة فقط. |
| **Reversal Entry** | قيد عكسي | الطريقة الوحيدة لتصحيح قيد مالي بعد `hard-close` — إنشاء قيد معاكس في فترة مفتوحة. |
| **Append-Only Ledger** | دفتر أستاذ للإضافة فقط | `FinancialLedger` لا يقبل `UPDATE` أو `DELETE` — فقط `INSERT`. التصحيح بقيود عكسية. |
| **Moving Average Cost (WAC)** | متوسط التكلفة المرجّح | طريقة تسعير المخزون — يُحدَّث تلقائياً عند كل استلام جديد. |

## المصطلحات الزراعية

| المصطلح | الترجمة | التعريف |
|---------|---------|---------|
| **CropPlan** | خطة المحصول | خطة موسمية لمحصول في موقع محدد — تتضمن الميزانية والعائد المتوقع والوصفة المعيارية. |
| **CropRecipe (BOM)** | الوصفة المعيارية | قائمة المواد المعيارية لمحصول معين حسب المرحلة الفينولوجية — الأساس لكشف الانحرافات. |
| **BiologicalAssetCohort** | دفعة أصول بيولوجية | مجموعة أشجار مزروعة في نفس الفترة — تُتتبع بشكل مستقل عمرياً ومالياً. |
| **Phenological Stage** | المرحلة الفينولوجية | مرحلة نمو المحصول (إنبات، إزهار، إثمار، حصاد) — تحدد المواد المسموحة. |
| **Dipstick (السيخ)** | قياس المقياس الفيزيائي | قراءة مستوى الوقود الفعلي في الخزان — تُقارن بالاستهلاك النظري لكشف التلاعب. |
| **Ratooning** | التجديد الخضري | إنتاج محصول جديد من جذور المحصول السابق (مثل الموز) — يُتتبع عبر `parent_cohort_id`. |

## المصطلحات التقنية

| المصطلح | التعريف |
|---------|---------|
| **RLS (Row-Level Security)** | سياسات أمان على مستوى الصف في PostgreSQL — تمنع استعلامات بيانات مزرعة أخرى حتى لو تجاوز الكود. |
| **Idempotency (Cache & Replay)** | إعادة إرسال نفس الطلب لا تُنشئ عملية مكررة — يُعاد الرد المُخزّن من العملية الأولى. |
| **Fiscal Period** | فترة مالية شهرية — حالات: `open` → `soft-close` → `hard-close`. |
| **Zombie Table** | جدول موجود في قاعدة البيانات لكن غير مُعرّف في Django models. |
| **Ghost Column** | حقل مُعرّف في Django لكن غير موجود في قاعدة البيانات (migration غير مُطبّق). |
| **Air-Gapped Deployment** | نشر في بيئة مُنقطعة عن الإنترنت — التحديثات عبر وسائط تخزين قابلة للإزالة. |
| **Shadow Mode** | وضع المراقبة — يسمح بالعمليات المالية لكن يُخفي الواجهات المالية المعقدة عن المستخدم العادي. |
| **Strict Mode** | الوضع الصارم — يعرض كل الوحدات المالية ويفرض جميع القيود. |

## المصطلحات الشرعية

| المصطلح | التعريف |
|---------|---------|
| **Zakat (الزكاة)** | الالتزام الشرعي على المحاصيل. نسبة `10%` للزراعة المطرية، `5%` للزراعة بالآبار، `7.5%` للري المختلط. |
| **IAS 41** | المعيار المحاسبي الدولي للزراعة — يتطلب إعادة تقييم الأصول البيولوجية بالقيمة العادلة. |
