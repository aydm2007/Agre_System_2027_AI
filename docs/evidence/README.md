# Evidence Storage Policy

جميع أدلة الجاهزية النهائية يجب أن تحفظ هنا أو تحت `docs/evidence/closure/`.

القواعد:
- لا تحفظ أدلة الجاهزية النهائية داخل `/tmp` فقط.
- كل تشغيل يجب أن ينتج ملفًا مسمى بتاريخ ووقت.
- كل ملف يجب أن يصنف النتيجة: PASS أو FAIL أو BLOCKED.
- لا تُرفع الدرجة النهائية إلى 98–99 بدون أدلة PostgreSQL + اختبارات + runtime snapshots محفوظة هنا.
