# Readiness Report Index

## Purpose

This index prevents reference inflation by separating the current governing readiness evidence from historical reports.

## Governing current source

- Current live score authority:
  - `docs/evidence/closure/latest/verify_axis_complete_v21/summary.json`
- Human-readable live evidence index:
  - [../evidence/EVIDENCE_SUMMARY_V21.md](../evidence/EVIDENCE_SUMMARY_V21.md)

## Reading rule

- If a historical report claims `100/100` more broadly than the current evidence standard allows, the historical report is informative only.
- The latest canonical summary always overrides older broad readiness wording and any fixed readiness report.
- Local ACL-locked `*.codex.tmp` residues are non-governing local artifacts when they are ignored and absent from canonical evidence.
