# التقارير التنفيذية العربية – V6

## النطاق
حزمة لوحات تنفيذية عربية RTL للإدارة العليا والإدارة المالية والرقابة.

## المؤشرات الإلزامية
- السيولة التشغيلية
- الانحرافات الحرجة
- التكلفة مقابل الخطة
- الجاهزية الشهرية للإقفال
- العهد المفتوحة
- الموردون المتأخرون
- المخاطر الموسمية
- الالتزام بالموافقات

## المعايير
- اللغة العربية أصلًا لا ترجمة لاحقة فقط.
- دعم اتجاه RTL والخطوط العربية والتوقيت المحلي.
- عدم إظهار بيانات حساسة أعلى من صلاحيات المستخدم.
