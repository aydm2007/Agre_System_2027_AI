# Agri Asset ERP – الدليل العربي المختصر (V6)

هذه الحزمة تمثل **V6 Arabic Enterprise Expansion Candidate**.

## ما الجديد في V6
- عقد مركزي للتدقيق الحساس عبر `AuditEventFactory`.
- توسعة تشغيلية متعددة المواقع وOffline-first.
- حزمة تقارير تنفيذية عربية RTL.
- توسيع doctrine والعمليات العربية المؤسسية.
- بوابات static جديدة تغطي التوسعة الإلزامية.

## أوامر مهمة
- `make verify-v6-static`
- `bash scripts/ops/preflight_enterprise.sh`
- `make enterprise-up`

## ملاحظة صريحة
ما يزال ختم 100/100 Evidence-Gated إلى حين تشغيل بيئة Django/DB/Frontend كاملة وإثبات الاختبارات الحية.
